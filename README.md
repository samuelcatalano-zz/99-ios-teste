# 99teste

#### Requisitos de produto:

* Gostaria que fosse possível se cadastrar no app.
* Como um passageiro gostaria de pedir um motorista, podendo informar onde ele deve me buscar.
* Gostaria de ver os motoristas por perto em tempo real.
* (Bonus): tornar a app resiliente para que o usuário tenha uma experiência incrível. 
* (Super Bonus) Criar uma experiência de pedir motorista diferente da app atual.


#### Requisitos técnicos (iOS)

* Linguagem Objective-C
* No minimo 2 views controllers.
* Desenvolver na ultima versão do target do iOS.
* O cadastro do usuário deve ser persistido.
* A utilização de bibliotecas de terceiros é permitido.
* Testes unitários.
* Testes de interface.
* (Bonus) Utilizar controle de dependência (Cocoapods ou Carthage)
